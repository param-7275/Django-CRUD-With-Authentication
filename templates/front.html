WELCOME TO MINI AUTHENTICATION SYSTEM WITH CRUD OPERATION-

Its a mini login,signup and logout page whereas the data of the student is stored in a table form .

ONLY AUTHENTICATED USER IS SHOW THE RECORD:

before the show record of student you need to signup if you already signup then simply click
on the login buton and enter the students record and check the data of all the students.

CREATOR: PARAMJEET SINGH -

SUMMARY:-

Its a simple authentication system with using CRUD(Create, Read, Update, Delete) operation
and Authentication (login logout & signup) system using Django, Python, Html and CSS.
In this project store a data of students when the person login in form then all data show in
the front then he can edit, delete, create and update the data only when the user is authenticated.
